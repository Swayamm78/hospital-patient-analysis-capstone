# Hospital Patient Data Analysis

Capstone Project| KIIT University | Roll No: 23052684 | April 2026

 📋 Project Overview
Analysis of hospital patient records to identify disease patterns, doctor workload distribution, and treatment cost insights.

🎯 Scenario
Hospital management wants to understand:
- Which diseases are most common?
- Which doctors treat the most patients?
- What is the average treatment cost?

📊 Deliverables

 1. Interactive Dashboard (`hospital_dashboard.html`)
- Disease distribution pie chart
- Doctor workload bar chart
- Patient records table
- Cost analysis & key insights

 2. SQL Analysis (`hospital_queries.sql`)
- Display all patient records
- Count patients by disease
- Calculate average treatment cost
- Identify busiest doctor

 3. Python Analysis (`analysis.py`)
- Load and analyze dataset
- Disease frequency calculation
- Treatment cost statistics
- Doctor workload grouping
- Matplotlib visualizations

 4. Excel Workbook (`Hospital_Analysis.xlsx`)
- Raw patient data sheet
- Pivot tables (Disease count, Doctor workload)
- Embedded charts
- Cost summary statistics

🔍 Key Findings
- **Most Common Disease**: Diabetes (4 patients, 33%)
- **Busiest Doctor**: Dr Shah (5 patients)
- **Average Cost**: ₹7,500
- **Cost Range**: ₹1,500 - ₹22,000
- **Most Expensive**: Heart Disease (avg ₹20,500)

💻 Tech Stack
- **SQL**: MySQL/PostgreSQL
- **Python**: pandas, matplotlib, openpyxl
- **Frontend**: HTML5, CSS3, Chart.js
- **Excel**: Pivot tables, embedded charts
